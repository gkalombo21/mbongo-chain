# Mbongo-Chain Developer Rewards Program

## Overview
The Mbongo-Chain project recognizes the critical role early developers play in building our hybrid blockchain. To acknowledge and retain top contributors, a portion of MBG tokens is reserved for early builders and reviewers once the network launches. This program is designed to reward meaningful contributions across AI, blockchain consensus, DAO tooling, documentation, and community growth.

## How It Works
1. **Contribution Logging:** Each submission (code, documentation, research) is recorded through GitHub pull requests and issues.
2. **Verification:** Core maintainers review contributions for quality, security, and alignment with project goals.
3. **Allocation:** Approved contributions are assigned provisional MBG amounts and documented in `contributors.json`.
4. **Distribution:** Once Mbongo-Chain mainnet launches, MBG allocations are released according to vesting schedules (e.g., for core and advisory roles).

## Roles & Rewards
| Role | Description | Example MBG Allocation |
|:--|:--|:--|
| **Core Developer** | Leads features, maintains modules, responds to reviews | 50,000 MBG |
| **Contributor** | Implements features, fixes bugs, improves docs/tests | 10,000 – 25,000 MBG |
| **Advisor** | Provides strategic guidance, audits code | 15,000 MBG |
| **Community Champion** | Drives adoption, organizes events, supports users | 5,000 MBG |

*Allocations are illustrative and subject to DAO approval.*

## Transparency & Tracking
- All allocations are tracked in `internal/contrib/contributors.json`.
- Each record includes contributor details, role, joined date, status, and reserved MBG.
- Updates to the file occur alongside PR merges or governance decisions.

## Join the Project
1. **Fork the repository** and set up your development environment following `CONTRIBUTING.md`.
2. **Pick an issue or propose a feature** aligned with Mbongo-Chain’s roadmap.
3. **Submit a pull request** detailing your work and tests.
4. **Engage with reviewers** to ensure quality and security standards are met.
5. **Track your impact**—once merged, your contributions are logged and you become eligible for MBG allocations upon mainnet launch.

We look forward to building Mbongo-Chain together—your expertise helps fuel decentralized AI compute for the world.
